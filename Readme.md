# Aslam - Warehouse 3D Management System

Sistema profissional de gerenciamento de estoque com visualização 3D interativa, autenticação multi-nível e validações de negócio.

## Características

- ✅ Visualização 3D interativa de prateleiras (Three.js)
- ✅ Sistema de autenticação com 3 níveis de privilégio
- ✅ Validação de volume por prateleira
- ✅ Gerenciamento de SKU com código de barras
- ✅ API REST robusta em Go
- ✅ Testes automatizados
- ✅ Deploy em Render com Docker
- ✅ Suporte a Mouse + Touch

## Stack Técnico

**Backend:**
- Go 1.23+
- Gin Framework
- PostgreSQL
- sqlc (type-safe SQL)

**Frontend:**
- React 19 + TypeScript
- Vite
- Three.js
- TailwindCSS
- Zustand

## Requisitos

- Go 1.23+
- Node.js 20+
- Docker
- PostgreSQL 14+
